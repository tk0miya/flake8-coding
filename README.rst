Checks for coding magic comment (``coding:``) in all python files.

Install
--------

Install with pip::

    $ pip install flake8-coding

Further options
----------------

accept-encodings
    Acceptable source code encodings for ``coding:`` magic comment.
    Default is ``latin-1, utf-8``.

Requirements
-------------

* Python 2.6, 2.7, 3.3, 3.4
* flake8
